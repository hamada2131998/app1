
# نظام Cash & Custody لإدارة الكاش والعهدة

## بيئات العمل (Environments)
يدعم المشروع الآن فصل الإعدادات بين التطوير والإنتاج:
- **التطوير المحلي:** استخدم `.env.local` بناءً على نموذج `.env.local.example`.
- **الإنتاج:** استخدم `.env.production` بناءً على نموذج `.env.production.example`.

## النشر (Deployment)
للحصول على تعليمات النشر التفصيلية، يرجى مراجعة [دليل النشر](docs/deploy.md).

## الأوامر الأساسية
- `npm run dev`: تشغيل بيئة التطوير.
- `npm run build`: بناء النسخة النهائية للإنتاج.
- `npm run check`: تشغيل البناء واختبارات الـ Smoke معاً للتأكد من سلامة الكود قبل النشر.
- `npm run test:smoke`: تشغيل اختبارات Playwright الآلية.
