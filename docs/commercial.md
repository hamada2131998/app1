
# النطاق التجاري (Commercial & Scope) - الإصدار التجريبي

### ما يشمله الاشتراك (In-Scope)
- استخدام النظام لإدارة الكاش والعهد بشكل غير محدود.
- استضافة البيانات والمرفقات (حتى 10GB شهرياً).
- التحديثات الأمنية وإصلاح الأخطاء البرمجية.

### خارج النطاق (Out-of-Scope)
- **تخصيص دورة العمل:** أي تغيير في مسار (Draft -> Submitted -> Approved).
- **التكاملات الخارجية:** الربط مع أنظمة ERP أو الربط مع البنوك (API).
- **التقارير المخصصة:** تصميم تقارير خارج القوالب الموجودة.

### سياسة طلبات التغيير
- أي طلب خارج النطاق يتطلب "بيان عمل" (SOW) مستقل وتقييم تكلفة إضافية.
