
# وثيقة تجميد النظام (System Freeze Policy) - V1.0.0-beta.1

تعتبر هذه النسخة هي "المرجع المستقر" لإطلاق العميل الأول.

### 1. الأجزاء المجمدة (Frozen Components)
- **قاعدة البيانات (Schema):** لا يسمح بأي تعديل على أسماء الجداول أو العلاقات (FKs).
- **سياسات الأمان (RLS):** تم تثبيت سياسات عزل الشركات ولا تعدل إلا بقرار أمني.
- **دورة الاعتماد (Workflow):** المسار (Draft -> Submitted -> Approved) ثابت برمجياً.
- **التكامل مع Supabase:** إصدار مكتبة `@supabase/supabase-js` مثبت على 2.45.0.

### 2. التغييرات المسموحة (Permitted Changes)
- إصلاح الثغرات الأمنية (Critical Bugfixes).
- تعديل نصوص واجهة المستخدم (UI Labels) لتحسين تجربة الاستخدام.
- إضافة فهارس (Indexes) جديدة إذا ظهر بطء في الأداء عند زيادة البيانات.

### 3. آلية التحديث القادم
- أي ميزات جديدة (مثل تعدد العملات أو تصدير Excel) ستؤجل للإصدار `1.0.0-beta.2` أو الإصدار المستقر النهائي.
