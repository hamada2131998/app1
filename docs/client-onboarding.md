
# دليل تهيئة العميل (Client Onboarding Guide)

مرحباً بك في نظام **Cash & Custody**. اتبع هذه الخطوات لتهيئة شركتك للعمل الفعلي.

### 1. بيانات الدخول الأولى
- ستصلك دعوة عبر البريد الإلكتروني من النظام.
- قم بتعيين كلمة مرور قوية (8 خانات على الأقل).
- سجل دخولك بصفتك "المدير العام" (OWNER).

### 2. خطوات الإعداد الذاتي (إلزامي)
يجب على العميل القيام بالخطوات التالية قبل بدء العمليات المالية:
- [ ] **تعريف الفروع:** من صفحة الإعدادات، أضف فروعك (مثل: المعرض الرئيسي، مستودع المنطقة).
- [ ] **الحسابات النقدية:** لكل فرع، عرّف "خزينة نقدية" أو "حساب بنكي" (مثل: عهدة الفرع، حساب الراجحي).
- [ ] **دعوة الموظفين:** أرسل دعوات للمحاسبين والموظفين وحدد أدوارهم بدقة.

### 3. تحذيرات هامة
- **الإقفال اليومي:** لا تقم بإقفال اليوم إلا بعد التأكد من جرد الخزينة فعلياً؛ فالإقفال يمنع التعديل الرجعي.
- **الحركات المعتمدة:** بمجرد اعتماد المحاسب للسند، لا يمكن حذفه أو تعديله إلا عبر "سند تسوية" جديد لضمان نزاهة السجلات.
