
# دليل الدعم الفني (Support Playbook)

### الأسئلة الشائعة (FAQ)
1. **لماذا لا أستطيع تعديل السند؟**
   - السندات بحالة "معتمد" (Approved) مقفلة برمجياً. يجب عمل سند عكسي للتصحيح.
2. **الموظف لم تصله دعوة البريد؟**
   - اطلب منه تفقد صندوق "الرسائل غير المرغوب فيها" (Junk/Spam).
3. **كيف أحذف فرعاً؟**
   - لا يمكن حذف فرع مرتبط بحركات مالية؛ يمكنك فقط "تعطيله" لمنع استخدامه مستقبلاً.
4. **هل تظهر مسوداتي للمحاسب؟**
   - لا، المسودات خاصة بك. يجب الضغط على "إرسال للاعتماد" لتظهر للمحاسب.
5. **ما هو الحد الأقصى للمرفقات؟**
   - 10 ميجابايت لكل ملف، ويفضل استخدام صيغ PDF أو الصور المضغوطة.
